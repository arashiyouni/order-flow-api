## 📦 Instalación de dependencias

```bash
$ npm install
```

### 🐋Inicialiar Docker

```
docker compose up -d
```

### 🌱 Correr creacion de seed

```
npm run setup
```

### 🔌 Enlace de conección de mongo

```bash
mongodb://localhost:27017/prisma-mongo?authSource=admin&retryWrites=true&w=majority
```
